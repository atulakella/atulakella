# Atul Akella

Final-year ECE at VIT-AP. I work at the hardware-software boundary, firmware, Linux drivers, and hardware accelerators. Most problems get interesting when you stop trusting the abstraction.

---

### currently

**[real-tim-shady](https://github.com/atulakella/real-tim-shady)** — Full heterogeneous SoC bring-up on the STM32MP157D-DK1. Custom Yocto BSP layer, PREEMPT_RT Linux `6.6.116-rt66`, bidirectional RPMsg IPC over OpenAMP, and a custom `rpmsg_driver` kernel module. A motor control application that demands hard real-time on one core and Linux orchestration on the other, built from the ground up to make both work together.
